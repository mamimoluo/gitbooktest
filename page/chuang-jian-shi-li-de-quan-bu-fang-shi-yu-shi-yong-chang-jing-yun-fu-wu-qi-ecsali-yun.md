# 创建实例的全部方式与适用场景-云服务器 ECS-阿里云

本文介绍创建 ECS 实例的几种方式，以及各自的适用场景，您可以按需选择任意一种实例购买和创建方式。

| 方式                                                                            | 适用场景                                                                       |
| ----------------------------------------------------------------------------- | -------------------------------------------------------------------------- |
| [快速购买实例](/broken/pages/6f65bd1a2136bd17ce342809755aae9a8b459581)              | 快速购买提供了四种配置多种套餐的实例规格。您可以在几分钟内，以最简单的方式购买一个ECS实例，提高配置效率。                     |
| [自定义购买实例](/broken/pages/b6b33d4cf4d7c96e4733f8ff6424cc84d1e794fa)             | 自定义购买可根据业务场景灵活地选择实例付费类型、镜像类型、实例规格、存储、带宽、安全组等配置，满足您的业务需求。                   |
| [使用自定义镜像创建实例](/broken/pages/1d2f0a8362761efeb130879ed2b341690d3fcc01)         | 如果您有习惯使用的操作系统、应用程序等配置，您可以先创建自定义镜像，通过自定义镜像创建实例，提高配置效率。                      |
| [使用社区镜像创建实例](/broken/pages/37f4267c9943eb10a358314fa0d34cb3d1ef9b07)          | 您可以使用社区镜像快速部署与业务需求匹配的操作系统、应用程序和数据的ECS实例，提高配置效率。                            |
| [购买相同配置的实例](/broken/pages/e9b61459cf86c31e8d80cf505aae4dafd20a62be)           | 如果您需要一台和已有实例相同配置的实例，可直接创建相同配置的新实例，提高在特定场景下横向扩展的效率。                         |
| [克隆实例](/broken/pages/39a954e097dbbacfaf635939dced025718d52377)                | 克隆实例可以快速创建与源实例相同或相似的实例，一般用于复制测试环境、扩展应用程序、容灾备份、迁移和升级等场景，可以提高业务的可靠性、可扩展性和效率。 |
| [使用实例启动模板创建实例](/broken/pages/f7f372997be2da534c2155fa8429fc2ab8dbcc86)        | 提前创建启动模板，设置好模板后能实现一键创建，更加方便快捷。                                             |
| [使用弹性供应组API批量创建ECS实例](/broken/pages/b64794b520e30be8df0c5f85ee58f8ed613bfa94) | 在需要大量创建按量付费实例的情况下，您可以通过调用API接口完成创建操作更为高效。                                  |
